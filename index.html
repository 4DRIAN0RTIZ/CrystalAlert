Quick Installation

Get started in under a minute.

<!-- CSS -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/4DRIAN0RTIZ/CrystalAlert@v1.0.0/dist/crystal-alert.min.css">

<!-- JS -->
<script src="https://cdn.jsdelivr.net/gh/4DRIAN0RTIZ/CrystalAlert@v1.0.0/dist/crystal-alert.min.js"></script>

<!-- Optional: Dark theme -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/4DRIAN0RTIZ/CrystalAlert@v1.0.0/dist/themes/dark.min.css">
# Clone the repository
git clone https://github.com/4DRIAN0RTIZ/CrystalAlert.git

# Copy dist files (minified)
cp CrystalAlert/dist/crystal-alert.min.js your-project/
cp CrystalAlert/dist/crystal-alert.min.css your-project/

# Copy themes (optional)
cp -r CrystalAlert/dist/themes your-project/

API Reference

Simple methods for all your notification needs.

Crystal.fire(options)

Display a modal alert dialog. Returns a Promise.

  • title The alert title text
  • text Plain text content
  • html HTML content (overrides text)
  • icon success | error | warning | info
  • iconHtml Custom HTML for icon
  • showCloseButton Show X button in corner
  • showCancelButton Show cancel button
  • preConfirm Async function before confirm
  • onOpen / onClose Lifecycle callbacks

Crystal.toast(options)

Display a non-blocking toast notification.

  • title The toast title text
  • text / html Content (plain or HTML)
  • icon / iconHtml Built-in or custom icon
  • duration Auto-dismiss ms (0 = persistent)
  • position top-right | top-left | bottom-right | bottom-left

Crystal.setTheme(name)

Load a theme CSS file. Returns a Promise.

  • name default | dark | minimal | custom
  • returns Promise that resolves when loaded
// Load dark theme
await Crystal.setTheme('dark');

// Custom theme path
Crystal.setThemePath('/css/themes/');
await Crystal.setTheme('brand');
